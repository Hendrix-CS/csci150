CSCI 150 - Lab 13
Civic Data Hacking

Instructions

  • First, you will need to create a free Microsoft Azure Notebook account. Visit this webpage to start the process.
  • Click on the "Sign in" text in the upper-right corner of the page. Log in with your Hendrix email account and password.
  • Next, visit this webpage, and click on the green "Clone" button in the upper-right corner of the screen.
  • This should make a copy of the project for you in your projects folder. Now click on "My Projects" and the "Civic Hacking Lab" section.
  • In your copy of the project, click on the "Lab 13 – Civic Hacking.ipynb" file in the project listing. This should open the notebook in the Azure Jupyter system in a separate browser tab, and you can start editing the file with your code. Follow the instructions in the file to complete this lab.
  • You can save your progress using "Ctrl-S" or "Cmd-S", and then return to this file any time.
  • To submit your file for grading when you are finished, return to the "Civic Hacking Lab" folder listing in the "My Projects" tab. Select the circle to the left of the "Lab 13 – Civic Hacking.ipynb". Then, click on the down arrow icon in the menu, to the left of the trash can icon. This will allow you to download the file onto your computer; then you can upload it using the usual turnin form.
