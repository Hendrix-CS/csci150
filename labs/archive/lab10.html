Overview

In this lab, we will be learning to use and reason about components in hierarchies, through programming mobile phone apps. We will use App Inventor, an easy-to-program system that returns us once again to block programming similar to Scratch and ModKit.

Step 1: Setup

To use App Inventor, you will need two pieces: an Android mobile phone, and a Google account. If you or your partner do not have an Android mobile phone, we will have some available for use during this lab.

You will need to connect your phone to WiFi, and have the MIT AI2 Companion App installed on your phone.

Follow the Setup Instructions on the App Inventor website, choosing Option 1.

Step 2: Hello Purr

Each app we make will involve two main parts, using the App Inventor Editor.

First, we will be adding components and setting up the layout for the visual interface in the Designer window. In every project, you will first find this Designer window with four main parts. On the left is the Palette, which lists off all the possible components you could add to your apps. The Viewer displays the GUI and visual structure of your app. Next is the Components hierarchy, showing the has-a structure of your app. Finally, the Properties will show the individual attribues of each component and allows you to edit them. The general structure of making the design is to drag components from the Palette onto the Viewer, then select them in the Components hierarchy and edit their attributes.

Second, we can then add the functionality for these components through writing code in the Blocks window. There are many built-in blocks available in every app, and the components we specifically dragged out of the palette will also have blocks that show up in the Blocks window on the left. To add functionality to your programs, drag out the blocks you need into the Viewer window and assemble them into your program.

Your first app to make is called HelloPurr, in which you will create a button with an image of a cat, and then tap the button to make it meow.

Explore

Add another image button to your app and a new associated sound, so the user can either tap on the cat or your new image.

Mole Mash is a more complicated app using the Canvas and Sprites to create a simulation of the carnival game Whack-A-Mole.
