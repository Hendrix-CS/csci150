Description

For this project, we will explore more complex interactions with the user by implementing control statements and loops. You will create a program for users to play a word game of your choice.

In particular, your project should include most of the following concepts we have discussed in class:

  • Input / Output
  • Math
  • Strings, Integers, Floats and Booleans
  • Control Statements
The game can be for one person against a computer, or two players against each other.

Some examples are

  • Fortune Teller
Do not feel limited by the suggestions on this list. Find something that interests you and implement it. If you choose something else, check with us first, so we can advise you on the scope of your project.

You should explain your program to the user, being clear exactly what input is expected and valid whenever requesting information from them.

The user must be able to enter appropriate information when requested, and the input they enter must have an impact on the output of the program.
